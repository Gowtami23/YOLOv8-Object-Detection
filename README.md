# 🎯 YOLOv8 Real-Time Object Detection

## 📌 Overview
This project implements a real-time object detection system using YOLOv8.
It supports detection on images, videos, and live webcam streams.

---

## 🚀 Features
- Image detection
- Video detection
- Live webcam detection
- Real-time bounding boxes with confidence scores

---

## 🛠 Tech Stack
Python | YOLOv8 (Ultralytics) | OpenCV | NumPy

---

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run detection:
python yolo_detect.py
